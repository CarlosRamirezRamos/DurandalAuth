Welcome to the DurandalAuth template

This template provide a way to authenticate and authorize users similar to the default ASP MVC 4 Internet template. You have Forms and oAuth authentication built-in with the main social providers among another useful features

Try to sign in for a new account using the membership system or the oAuth providers.

A role User will be asigned to any new user and will permit access to the User view. If you don´t want to sign in, you can use the existing user user/user1234

If you want to access the Admin view, you need the role Administrator so you can only access this view with an admin user. Try to log in with the following one admin/admin1234

Important!!

The authorization system in the client can be always be hacked, so remember to always authenticate and authorize the request in the server when retrieving the necessary data for your views
