Welcome to the Durandal auth authentication template

This template pretend to be a port from the default ASP MVC 4 Internet template, providing forms and oAuth authentication with the main providers

Try to sign in for a new account using the membership system or the oAuth providers.

A role "User" will be asigned to any new user and will allow access to the User view.

If you want to access to the Admin view you need the role "Administrator" so you can only acccess to it with an admin user. Try to log in with the followind one admin/admin1234

Important !!

The authorization system in the client can be always be hacked, so remember to always authenticate and authorize the request in the server when retrieving the necessary data for your views
