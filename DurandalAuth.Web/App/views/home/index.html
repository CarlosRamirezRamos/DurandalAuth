Welcome to the DurandalAuth 2.0 template

This template provides a way to authenticate users using Durandal and token authentication with the new ASP.NET Identity membership.

Visit the project repo on Github

Breeze for data management

Client-side data management uses the excellent Breeze JS framework.

ASP MVC and Web API

ASP MVC and Web API are used on the server-side.

Unit of Work and Repositories

You will find an implementation of this pattern on both the server (Entity Framework) and the client (Breeze).

SEO Optimization

The template is ready to be SEO crawlable using a headless browser

ASP.NET Identity

This template uses the new ASP.NET Identity system and token authentication.
